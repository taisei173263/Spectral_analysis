# Submission Notes

## 前処理

本パイプラインでは、近赤外スペクトルの散乱・ベースライン変動に対応するため、以下の前処理を設定で切り替えます。

- `raw`: 生スペクトル。
- `snv`: サンプルごとの平均・標準偏差で標準化。
- `msc`: 学習データ平均スペクトルを参照にした散乱補正。
- `savgol`: Savitzky-Golay平滑化および1次/2次微分。

水分に関連しやすい 6600-7200 cm^-1、5000-5400 cm^-1、4500-4900 cm^-1 の統計量も特徴量化します。

## 学習

主評価は未知樹種への汎化を意識したGroupKFoldです。モデルは、PLS/Ridge/KernelRidgeのようなケモメトリクス寄りのモデルと、LightGBM/CatBoost/ExtraTreesなどの非線形モデルを比較し、OOF性能に基づいて重み付き平均します。

## 生成AI利用申告

本実装では、Cursor上の生成AI支援を利用して以下を作成しました。

- EDAノートブックとEDAレポート生成スクリプト。
- 前処理、特徴量生成、CV、モデル学習、アンサンブル、提出CSV生成コード。
- 実験設定、ジョブスクリプト、実験記録ドキュメント。

データ自体の改変、外部データ追加、目的変数のリークを意図した処理は行っていません。
